title: How the council handles personal information (Brighton & Hove)
tags: [governance, PII, personal-information, data-handling]
summary:
- Council collects and uses personal information to provide services.
- Personal data is protected and used only for specified purposes.

key_points:
- Personal information collected only when necessary for services.
- Data used for the purpose it was collected for.
- Information shared with other agencies only when legally required or for service delivery.
- Data kept secure and only accessible to authorised staff.
- Personal information retained only as long as needed.
- You can request to know what personal data the council holds about you.
- Council privacy notice explains how your data is used.
- Contact data protection officer with concerns about personal data use.
